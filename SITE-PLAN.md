# Plan: The Sandlot Tribune — Full GitHub Pages Website

## Context
The sandlottribune GitHub Pages repo currently hosts only a bare-bones article index. The goal is to transform it into a complete, professional-looking website matching the Canva design at sandlottribune.my.canva.site — warm tan/cream palette, Marcellus serif typography, top navigation with 5 sections, and a news hub home page. Under-construction sections let readers know what's coming. New single-file articles (generated by TSDL2026 pipeline) replace the old split-article format.

## Design System (assets/css/main.css)

### Colors
```css
--bg:           #F6F3EC   /* cream — page background */
--surface:      #ECE4DA   /* light tan — nav, cards */
--border:       #B9A590   /* medium tan — dividers, card borders */
--text-primary: #36302A   /* dark brown — headings, body */
--text-secondary: #574C3F /* dark tan — bylines, labels */
--accent-red:   #BA2B2B   /* red — buttons, active nav, badges */
--accent-blue:  #1E3A5F   /* navy — links, hover states */
--overlay:      #36302A   /* dark brown — image overlays */
```

### Typography
- **All text**: `Marcellus` (Google Fonts) — headings AND body copy
- Fallback: `'Palatino Linotype', Georgia, serif`
- Load via: `<link href="https://fonts.googleapis.com/css2?family=Marcellus&display=swap" rel="stylesheet">`
- Size scale: hero 3.5rem → section headers 1.1rem uppercase → body 1rem → nav 0.9rem
- Note: Marcellus is a display/serif font — body line-height should be generous (1.7–1.8) for readability

### Components
- **Header/Nav**: `background: var(--surface)`, `border-bottom: 2px solid var(--border)`, fixed to top
  - Logo left, nav links right (flex layout)
  - Active link: underline in `var(--accent-red)` or `color: var(--accent-red)`
- **Article Cards** (home page): Image top, category badge, title (Marcellus), date, description, "Read →" link
- **Under Construction badge**: pill shape, `background: var(--accent-blue)`, white text, small uppercase
- **Section headers**: Marcellus, uppercase letter-spacing, `color: var(--text-secondary)`, thin bottom border
- **Footer**: minimal, cream background, dark brown text

## Page Specifications

### 1. index.html — Home / News Hub

**Layout (top to bottom):**
1. `<header>` — fixed nav with logo + 5 nav links (Home, Between The Chalk, Season Deadlines & Events, League Records, About / The Tribune)
2. **Hero** — site name large in Marcellus, short tagline ("The Official Publication of The Sandlot Dynasty League"), thin decorative rule
3. **Latest News** (h2: "LATEST") — Featured debut article + coming-soon placeholders
   - **ARI card** (full-width or prominent): featured image, "SEASON PREVIEW" badge, title, author, date, 1-sentence teaser, "Read →" link
   - **2 placeholder cards** styled with muted colors: category name + "More articles coming soon" — giving the grid a full look without fake content
4. **Browse by Section** (h2: "EXPLORE") — 2×2 grid of section cards
   - Season Previews → season-previews/index.html
   - Power Rankings → coming soon badge
   - Matchup Previews → coming soon badge
   - Trade Analysis → coming soon badge
5. `<footer>` — "© The Sandlot Tribune · The Sandlot Dynasty League" + season year

## CSS Architecture Note

All pages use `assets/css/main.css` for shared styles (header, nav, footer, cards, badges). Article HTML files generated by TSDL2026 keep their own embedded `<style>` blocks (team-specific colors). The shared CSS handles site chrome (nav, page wrapper, footer); article CSS handles article content. No conflict.

Path prefix for CSS links:
- Root-level pages (index.html, etc.): `assets/css/main.css`
- Subdirectory pages (season-previews/index.html): `../assets/css/main.css`
- Articles inside season-previews/: add nav header inline or skip nav in articles
